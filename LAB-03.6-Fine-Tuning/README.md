# 🤖 LAB-03.6 – Fine-Tuning

> Fine-Tuning a GPT-4.1 foundation model using Azure AI Foundry with a custom travel dataset.

## 📌 Objective

This lab demonstrates how to fine-tune a GPT-4.1 model using supervised learning in Azure AI Foundry. A custom JSONL travel dataset was used to train, deploy, and test a fine-tuned model with improved response consistency and task-specific behavior.

---

## 🚀 Tasks Performed

* Configured a supervised fine-tuning job.
* Uploaded a custom JSONL training dataset.
* Fine-tuned the GPT-4.1 foundation model.
* Automatically deployed the fine-tuned model.
* Compared the base model and fine-tuned model responses.
* Tested the deployed model in Azure AI Foundry Playground.

---

## 🛠️ Technologies Used

* Microsoft Azure AI Foundry
* GPT-4.1
* JSONL Dataset
* Azure AI Playground
* Supervised Fine-Tuning

---

## 📄 Report

The complete lab report with screenshots, observations, results, and conclusion is included in this repository.

---

## 🎯 Outcome

Successfully fine-tuned and deployed a GPT-4.1 model capable of generating more consistent and travel-focused responses based on the provided training dataset.
