# LAB 03.7 – Create and Deploy Prompt Flow using Azure AI Foundry

## Overview

This repository contains the report and implementation details for **Lab 03.6**, where a **Standard Prompt Flow** was created, configured, and executed in **Azure AI Foundry** using the **GPT-4.1** model. The lab demonstrates how to build prompt-driven AI workflows by connecting LLM nodes and generating dynamic AI responses.

## Objectives

* Create and configure a Standard Prompt Flow.
* Connect the workflow with an Azure OpenAI GPT-4.1 deployment.
* Execute the flow using dynamic input.
* Validate and generate AI-powered responses.

## Technologies Used

* Azure AI Foundry
* Azure OpenAI
* GPT-4.1
* Prompt Flow

## Repository Contents

* **LAB-C6 Report.pdf** – Complete lab documentation with screenshots, observations, results, and conclusion.
* **README.md** – Repository overview.

## Outcome

Successfully designed and executed a Prompt Flow in Azure AI Foundry, demonstrating prompt orchestration, LLM integration, and AI response generation using GPT-4.1.
