# Azure Foundry Lab 04 – Create an AI Hub in Azure AI Foundry

## Overview
This lab demonstrates the process of creating an AI Hub and Project in Azure AI Foundry. It covers project configuration, Azure resource provisioning, and verification of the deployed AI environment.

## Objectives
- Create an AI Hub and Project in Azure AI Foundry.
- Configure Azure subscription, resource group, and region.
- Deploy Azure AI resources successfully.
- Verify project endpoints and AI capabilities.

## Key Outcomes
- Successfully created and deployed an Azure AI Foundry Project.
- Configured Azure OpenAI services and project resources.
- Generated project endpoints and API credentials.
- Verified successful deployment and resource provisioning.

## Technologies Used
- Microsoft Azure
- Azure AI Foundry
- Azure OpenAI
- Azure Resource Manager
