# Lab 03.1 - Choose and Deploy a Language Model

## Objective

Explore Azure AI Foundry Model Catalog and compare language models using benchmark metrics and cost-benefit analysis.

## Activities Performed

- Opened Model Catalog
- Explored Benchmark Metrics
- Compared Multiple Language Models
- Analyzed Benchmark Metrics
- Performed Cost-Benefit Analysis
- Evaluated Model Leaderboard Rankings

## Models Evaluated

- GPT-4o
- GPT-4o Mini
- Grok 4.3

## Outcome

Identified GPT-4o as the preferred model for high-performance applications and GPT-4o Mini as the best balance between cost and quality.
