# LAB-03.1 - Choose and Deploy a Language Model

## Objective

To explore the Azure AI Foundry Model Catalog, compare language models using benchmark metrics, and identify the most suitable model for deployment based on performance, cost, and evaluation results.

## Activities Performed

* Explored Azure AI Foundry Model Catalog
* Reviewed model benchmark metrics and leaderboard rankings
* Compared GPT-4o, GPT-4o Mini, and Grok 4.3 models
* Analyzed Quality Index, Safety, Throughput, and Benchmark Cost
* Evaluated model performance using Azure AI Foundry Benchmarks
* Performed cost-benefit analysis for model selection
* Selected the most suitable model for deployment

## Deliverables

* Lab Report (PDF)
* Lab Report (DOCX)
* Model Comparison Analysis
* Benchmark Evaluation Results
* Observations and Conclusion
