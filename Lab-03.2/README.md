# LAB-03.2 : Deploy and Evaluate a Foundation Model

## Objective

To deploy a foundation model from the Microsoft Azure AI Foundry Model Catalog, test its capabilities in the Playground, and evaluate its performance using Azure AI Foundry's built-in evaluation framework.

## Activities Performed

* Explored the Azure AI Foundry Model Catalog
* Deployed the GPT-4.1 foundation model
* Tested prompts and responses in the Playground
* Configured a model evaluation workflow
* Executed evaluation using a sample dataset
* Analyzed evaluation metrics and generated results

## Deliverables

* Lab Report PDF
* Model Deployment Screenshots
* Playground Testing Results
* Evaluation Results
* Observations and Conclusion
